<h1>🎬 Netflix Power BI Dashboard</h1>

<h2>🎯 Project Goal</h2>
<p>
To analyze Netflix’s content library from various perspectives — including <strong>genre, country, average ratings, and viewer engagement</strong> — and present it visually to support fast and informed decision-making.
</p>

<hr>

<h2>📊 Key Features & Insights</h2>

<h3>🔀 Interactive Filters</h3>
<ul>
  <li>Toggle switch to filter metrics by:
    <ul>
      <li><strong>Movies</strong></li>
      <li><strong>TV Shows</strong></li>
      <li><strong>Combined View</strong></li>
    </ul>
  </li>
</ul>

<h3>📌 KPI Cards</h3>
<ul>
  <li>Show total content count at a glance:</li>
  <ul>
    <li><strong>5603 Titles</strong> in total</li>
    <li>2648 Movies</li>
    <li>2955 TV Shows</li>
  </ul>
</ul>

<h3>📉 Funnel Chart: Naming Group</h3>
<ul>
  <li>Visualizes title length categories (e.g., 2-word, 3-word titles)</li>
  <li>Helps analyze naming patterns and trends in content</li>
</ul>

<h3>🎭 Genre Analysis</h3>
<ul>
  <li>Combined <strong>Bar & Line Chart</strong> for each genre:</li>
  <ul>
    <li>Number of titles</li>
    <li>Average IMDb rating</li>
  </ul>
  <li><em>Insight:</em> Comedy and Drama dominate in volume, while Documentary and Biography lead in average ratings.</li>
</ul>

<h3>🌍 Country-wise Breakdown</h3>
<ul>
  <li>Tabular view with:
    <ul>
      <li>Total titles by country</li>
      <li>Average IMDb ratings</li>
      <li>Total viewer votes</li>
    </ul>
  </li>
  <li>Useful to identify strong regional content performers</li>
</ul>

<h3>🗺️ Map Visualization</h3>
<ul>
  <li>Interactive world map with bubble indicators</li>
  <li>Bigger bubbles = more content from that country</li>
  <li><strong>USA, India, and Canada</strong> rank among the top content producers</li>
</ul>

<h3>⭐ Top Title Previews</h3>
<ul>
  <li>Sidebar shows:
    <ul>
      <li>Netflix titles with images</li>
      <li>Plot summaries</li>
      <li>IMDb ratings & vote counts</li>
    </ul>
  </li>
  <li>Gives real-time visibility into trending shows and audience reception</li>
</ul>

<h3>📈 Overall Metrics</h3>
<ul>
  <li><strong>Average IMDb rating:</strong> 6.7</li>
  <li><strong>Total votes:</strong> Over 11 million</li>
</ul>

<hr>

<h2>🛠️ Tools & Skills Used</h2>
<ul>
  <li><strong>Power BI:</strong> Data modeling and dashboard creation</li>
  <li><strong>Power Query:</strong> Data transformation and cleaning</li>
  <li><strong>DAX:</strong> For calculated columns, custom KPIs like average ratings and filters</li>
  <li><strong>Data Storytelling:</strong> Focused on intuitive user experience and actionable insights</li>
</ul>
